About Household Finance AI

Back to Home

A smart financial management platform built as a final year project for Computer Science/IT students.

Project Overview

This web application demonstrates the integration of modern web technologies with artificial intelligence to create a comprehensive personal finance management system.

Key Features

AI Features
  • Smart transaction categorization
  • Anomaly detection
  • Predictive analytics
  • Natural language search
Financial Tools
  • Budget tracking
  • Expense visualization
  • Receipt scanning (OCR)
  • Financial insights

Technology Stack

Backend Development
  • Python Flask Framework
  • SQLAlchemy ORM
  • SQLite Database
  • RESTful API Design
Frontend Development
  • HTML5, CSS3, JavaScript
  • Bootstrap 5 Framework
  • Plotly.js for charts
  • Responsive Design
AI & Machine Learning
  • Google Gemini AI Integration
  • Tesseract OCR Engine
  • spaCy Natural Language Processing
  • Predictive Analytics
Additional Features
  • User Authentication System
  • File Upload & Processing
  • Data Export Capabilities
  • Real-time Notifications
Academic Project

This project is developed as part of the final year requirements for a Bachelor's degree in Computer Science/Information Technology. It demonstrates comprehensive understanding of full-stack web development, database design, and AI integration.
